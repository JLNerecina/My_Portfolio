## 💎 About the Project

This repository contains the source code for my interactive, highly-polished personal portfolio. Built with a modern tech stack and designed with a sleek **Cosmic Blue** dark theme, it serves as a central hub for my projects, milestones, skills, and open-source GitHub contributions. 

## 🚀 Features

- **Interactive 3D UI**: Immersive card elements using `react-parallax-tilt`.
- **Fluid Animations**: Smooth scrolling, page transitions, and staggered component rendering powered by `motion/react` (Framer Motion).
- **Live GitHub Integration**: Dynamically fetches and displays GitHub repository weights, top languages, and a contribution calendar utilizing the GitHub REST API.
- **Responsive Design**: Tailored for all screen sizes with CSS Grid and standard Tailwind mobile-first breakpoints.
- **Glassmorphism Aesthetics**: Deep dark backgrounds intertwined with subtle blue accents (`#3b82f6`) and blurred backdrop filters.

## 🛠️ Tech Stack

### Frameworks & Languages
- **React.js 18+** & **Vite**
- **TypeScript**
- **Tailwind CSS v4**

### Core Libraries
- `lucide-react`: For clean, scalable vector icons.
- `motion/react`: For advanced animations and gesture handling.
- `react-parallax-tilt`: For 3D floating hover effects.
- `react-github-calendar`: For GitHub activity visualizations.

## 🏃 Getting Started

To run this project locally, follow these steps:

1. **Clone the repository**
   ```bash
   git clone https://github.com/JLNerecina/portfolio.git
   cd portfolio
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```

4. **Open your browser**
   Navigate to `http://localhost:3000` to view the live preview.
